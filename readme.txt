== Description ==
Displays latest/critical/exploited vulnerabilities from ENISA EUVD via shortcodes, Gutenberg block, and widget.

== Shortcodes ==
[euvd_vulnerabilities type="latest|critical|exploited" count="5"]
[euvd_latest count="5"]
[euvd_critical count="5"]
[euvd_exploited count="5"]

== Installation ==
1. Upload folder to /wp-content/plugins/
2. Activate plugin
3. Configure settings under Settings -> EUVD Vulnerabilities
